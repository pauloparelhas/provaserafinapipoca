## TEMA CENTRAL: O que é um padrão?
**"Where can we find patterns?"** / Onde podemos encontrar padrões?

Padrões estão em TODOS os lugares ao nosso redor:
- Nos animais (animals) — manchas da onça, listras da zebra
- No supermercado (supermarket) — frutas organizadas por cor e tipo
- No futebol (football) — posições dos jogadores no campo
- Na nossa sala de aula (in our classroom) — cartazes, fileiras de carteiras
- Nas bibliotecas (libraries) — livros organizados na prateleira
- Nos livros (books) — páginas, capítulos, ilustrações
- No quarto (room) — objetos organizados

**Palavra-chave:** Pattern / Padrão — uma sequência que se REPETE seguindo uma regra.

---

## O QUE É UM ATRIBUTO? (What is an attribute?)

**Attribute** (em inglês) = **Atributo** (em português)

> "What something looks like." / "Como uma coisa parece."

Um atributo é uma CARACTERÍSTICA de um objeto. Os atributos que usamos em matemática são:

### 1. Color / Cor
Exemplo de padrão por cor: 🟢🔵🔴🟢🔵🔴🟢🔵🔴
- verde, azul, vermelho — repete!

### 2. Shape / Forma
Exemplos de formas: triângulo (triangle), quadrado (square), círculo (circle)
Exemplo de padrão por forma: △ ▭ ○ △ ▭ ○ △ ▭ ○

### 3. Number / Número
Exemplo de padrão por número: 4, 6, 6, 4, 6, 6, 4, 6, 6

### 4. Size / Tamanho
Exemplos: big (grande), small (pequeno)
Exemplo de padrão por tamanho: grande, pequeno, pequeno, grande, pequeno, pequeno...

### 5. Position / Posição
Exemplos: up (para cima), down (para baixo)
Exemplo de padrão por posição: ↑ ↓ ↑ ↓ ↑ ↓

### 6. Letters / Letras
Exemplo: P Q P Q P Q P Q P Q
- A letra que aparece se alterna — isso é um padrão!

## O QUE É O CORE (NÚCLEO)? (What is the core?)

**Core** (em inglês) = **Núcleo** ou **Parte que se repete**

> "The part that repeats." / "A parte que se repete."

O CORE é o menor grupo de elementos que se repete para formar o padrão.

### Como identificar o CORE:
Olhe a sequência: 🟣 🟣 🔺 🟣 🟣 🔺 🟣 🟣 🔺

Qual parte se repete? → 🟣 🟣 🔺 — esse é o CORE!

### Exemplos de CORE:
- Sequência: rosa, verde, azul, rosa, verde, azul → CORE = rosa, verde, azul (3 elementos)
- Sequência: círculo, triângulo, círculo, triângulo → CORE = círculo, triângulo (2 elementos)
- Sequência: P, Q, P, Q, P, Q → CORE = P, Q (2 letras)

## PERGUNTAS TÍPICAS DE PROVA

1. "What is the core of this pattern?" → Identificar a parte que se repete
2. "What comes next?" → Continuar a sequência
3. "What attribute is used?" → Identificar se é cor, forma, tamanho, etc.
4. "Create your own pattern using shapes." → Criar um padrão
5. "Circle the core." → Circular o núcleo na sequência
