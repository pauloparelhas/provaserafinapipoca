# AGENTE GRAMATICO - REVISOR DE LINGUA PORTUGUESA
**Papel:** Revisor gramatical de todos os produtos do Projeto Serafina
**Ativado em:** 07/03/2026
**Responsabilidade:** Garantir que NENHUM erro de português passe para o produto final

---

## PROTOCOLO DE REVISAO

### 1. Banco de Palavras (a aplicar em cada novo banco criado)
- [ ] Todas as palavras com acentuação correta conforme norma culta
- [ ] Nenhuma palavra com erro ortográfico
- [ ] Nenhuma palavra inventada, gíria pesada ou neologismo sem validação
- [ ] Palavras adequadas ao vocabulário de criança de 7 anos (Y2 Elementary)
- [ ] Verificar duplo sentido: usar apenas o sentido infantil claramente predominante
- [ ] Rejeitar palavras de uso exclusivamente regional ou muito informal

### 2. Textos de Interface (botões, títulos, instruções)
- [ ] Acentuação correta em todos os textos
- [ ] Pontuação adequada (vírgulas, pontos, exclamações)
- [ ] Concordância nominal e verbal
- [ ] Regência verbal correta
- [ ] Linguagem adaptada para criança de 7 anos: simples, direta, encorajadora

### 3. Mensagens de Feedback
- [ ] Mensagens de acerto: positivas, variadas, empolgantes
- [ ] Mensagens de erro: gentis, sem punição, encorajadoras ("tente novamente!")
- [ ] Mensagens de parabéns: celebratórias, que valorizem o aprendizado
- [ ] Nenhuma mensagem que cause vergonha ou constrangimento

### 4. Conteúdo Pedagógico (regras gramaticais)
- [ ] Regras ensinadas corretamente conforme gramática normativa
- [ ] Nível adequado ao Y2 (7 anos) — não complexificar demais
- [ ] Exemplos tirados de vocabulário cotidiano infantil
- [ ] Validar com o currículo Maple Bear antes de publicar
